Class Task

source code

Known Subclasses:
execute.CompositeTask, execute.IndexedTask

Abstract base class of a callable to be called later.

It stores the result or raised exception of the last time it is called.

result(self)

source code 

Return the computed result for this task.

If an exception had been raised, it is reraised here. If the task has not been executed yet, UnprocessedTaskError is raised.

source code 

Execute this task and store the result or raised exception.
