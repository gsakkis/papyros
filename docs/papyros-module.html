Package papyros

source code

A small platform independent parallel processing package.

papyros provides a simple uniform interface for executing tasks concurrently in multiple threads or processes, local or on remote hosts.

Convenience function for configuring a Logger with some basic options.

Mainly used internally by papyros, but can be also used by clients.
Parameters:
  • logger - The logging.Logger to be configured (papyros.log by default).
  • stderr - If True, log to standard error.
  • logfile - If not None, log to the given logfile. It can be combined with stderr=True.
  • mode - The write mode for the logfile; typically 'a' or 'w'.
  • level - The logger's level.
  • fmt - The format string to be passed to the logging.Formatter.
  • datefmt - The date format string to be passed to the logging.Formatter.
