                        PyroTaskDispatcherProxy

A proxy class of PyroTaskDispatcher.

Clients interact with instances of this class instead of actual PyroTaskDispatcher instances. The differences are that execute is invoked locally on the client and that close and run are not accesible remotely.

Method Details [hide private]

execute(self, tasks, timeout=None, ordered=False, chunksize=1, qsize=0)

source code 
Execute the given tasks.
Parameters:
  • tasks - An iterable of tasks to be executed.
  • timeout - An upper bound of how long is a task expected to take. Useful to prevent waiting for tasks that take forever to complete for whatever reason.
  • ordered - If True, the order of the yielded results matches the the order of the tasks that generated them. If False, the results' order is unspecified.
  • chunksize - The tasks iterable is chunked into slices of this size (or less) before being submitted to the dispatcher and workers.
  • qsize - The size of the underlying task queue.
Returns:
An iterator over the results of the executed tasks.
Raises:
  • TimeoutError - If a call to the returned iterator's next() exceeds the timeout.
