Module distributed

source code

A distributed process implementation of the papyros API.

Both the TaskDispatcher and each Worker is a separate process, running on one or more interconnected machines.

Requires: Pyro 3.7.

set_nameserver(host=None, port=None)

source code 

Specify the Pyro NameServer address.

If this function is not called, the NameServer is attempted to be discovered automatically by a broadcast lookup.
Parameters:
  • host - The hostname that is initially tried to find the NameServer on, or when the broadcast lookup mechanism fails.
  • port - The socket number on which the Name Server will listen for incoming requests.

register(obj, name, force=False)

source code 
Register obj to the PYRO_DAEMON with the given name.
Parameters:
  • obj - An ObjBase instance.
  • name - The fully qualified object's name. If it is a dotted name (e.g. 'my.object.name' it creates all necessary intermedieate nameserver groups ('my', 'my.object').
  • force - If False and there is already an entry for the given name, NamingError is raised. If True, the previous object is unregistered and obj is registered in its place.
