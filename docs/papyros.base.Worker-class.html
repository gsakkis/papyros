Class Worker

source code

object --+
         |
        Worker
Known Subclasses:
distributed.PyroWorker, multithreaded.MultiThreadedWorker, dummy.DummyTaskDispatcher

A Worker is assigned tasks from a TaskDispatcher, executes them and returns them to the respective TaskQueue that submitted the task.

This class is not directly usable; it must be subclassed to provide a way to call execute_task, typically in an infinite loop.
